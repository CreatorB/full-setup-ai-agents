# Agent Commands

Unified quick reference for three local coding agents on Windows.

## Daily Coding

- Aider: `adev`
- Hermes: `hdev`
- Pi: `pidev`

## Lower-Risk Daily Coding

- Aider: `asafe`
- Hermes: `hsafe`
- Pi: `psafe`

## Planning / Architecture

- Aider: `aplan`
- Hermes: `hplan`
- Pi: `pplan`

## Heavier Fallback

- Aider: `aheavy`
- Hermes: `hheavy`
- Pi: `pheavy`

## Flexible Model Switching

- Aider: `amodel <model> [path]`
- Hermes: `hmodel <model> [path]`
- Pi: `pmodel <model> [path]`
