# teaching-material

## Purpose

Generate teaching materials for IT/ICT (TIK) classes grades 7-12. Covers the Indonesian curriculum for computer science and information technology, including Office productivity, basic design, programming fundamentals, networking, and web development.

## LLM Required

Yes. Uses Ollama for content generation.

## Trigger

On-demand via terminal or Telegram chat.

## Supported Topics

### Grades 7-8 (SMP)
- Office productivity: Microsoft Word, Excel, PowerPoint
- Basic computer operations and digital literacy
- Internet safety and ethics
- Introduction to algorithms and logical thinking
- Basic graphic design (Inkscape, Canva)

### Grades 9-10 (SMP/SMA)
- HTML, CSS basics
- Introduction to programming (Scratch, Python basics)
- Spreadsheet formulas and data analysis
- Presentation design principles
- Basic networking concepts

### Grades 11-12 (SMA)
- Web development (HTML, CSS, JavaScript)
- Database fundamentals (SQL basics)
- Programming logic (Python, Java basics)
- Computer networking (IP addressing, subnetting, MikroTik basics)
- Cybersecurity awareness
- Introduction to mobile app development

## Output Formats

| Type | Description |
|---|---|
| Slide outline | Numbered topic outline with key points per slide |
| Quiz | Multiple choice, essay, and practical questions with answer key |
| Lab module | Step-by-step hands-on practical guide with objectives and evaluation |
| Worksheet | Fill-in-the-blank and short-answer exercises |
| Lesson plan | Full lesson plan with time allocation, methods, and assessment |

## Language

- Primary: Indonesian (Bahasa Indonesia)
- Technical terms: Bilingual ID/EN (e.g., "Jaringan Komputer (Computer Network)")
- Can generate full English versions on request

## Example Prompts (via Telegram)

```
"buat 10 soal pilihan ganda tentang HTML untuk kelas 10"
"buat modul praktikum Excel dasar untuk kelas 7"
"buat outline presentasi tentang Jaringan Komputer kelas 11"
"create a lesson plan for Introduction to Python, grade 9, in English"
"buat worksheet tentang subnetting untuk kelas 12"
"buat soal essay tentang keamanan internet kelas 8"
```

## Config

No additional configuration needed. Difficulty level is automatically adjusted based on the grade specified in the prompt.
